# healthy-habits

Browse health areas (sleep, cardiovascular, diet, and more), track the habits that move the needle in each, and see how those habits connect to key health indicators. Habit tracking and any custom habits you add are saved to your browser's local storage.

Live at **https://dylanweed.github.io/healthy-habits/** — automatically rebuilt and deployed on every push to `main` via GitHub Actions.

## Development

```
npm install
npm run dev
```
